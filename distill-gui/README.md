# distill-gui

**"Norton Commander for converting websites to clean Markdown"**

A simple, powerful desktop GUI for `distill` + `distill-render`.

## Features (Planned)

- Add URLs manually or via file upload
- Queue management
- Per-job options (JS render, stealth, delay)
- Real-time progress
- Rich export options (individual files, ZIP, single combined file, clipboard)

## Current Status

Working prototype with queue + options UI.

## Run

```bash
cargo run --release
```
