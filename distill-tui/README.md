# distill-tui

Cross-platform terminal UI for `distill_core`, with optional `distill-render` support for JS-heavy sites.

## Features

- Single URL and batch file execution
- Shared Markdown conversion through `distill_core`
- Optional render mode via `distill-render`
- Combined Markdown and ZIP export
- Works in a plain terminal on Windows, macOS, and Linux

## Run

```bash
cargo run
```

## Windows Icon

Drop `whiskey.png` or `whiskey.ico` into `distill-tui/assets/` before building on Windows to embed it into `distill-tui.exe`.
